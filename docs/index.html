How It Works

From Terminal to Visual

Pilotfish transforms complex ADB commands into simple visual interactions. Built with Electron and React, it provides a native desktop experience that bridges the gap between Android development tools and modern UI.

Key Architecture

Electron 40 + React 19
├── ADB Integration (@devicefarmer/adbkit)
├── Modern UI (shadcn/ui + TailwindCSS)
├── State Management (Zustand + TanStack Query)
├── Type Safety (TypeScript 5.9)
└── Testing (Vitest + Playwright)

Tech Stack

Electron 40 React 19.2 TypeScript 5.9 TailwindCSS 4 shadcn/ui Vite 7 TanStack Router TanStack Query Zustand Zod Vitest Playwright
